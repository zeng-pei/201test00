## 抽籤系統

這是一個帶有動畫效果的1到22號抽籤系統，使用Python實現。

### 功能特色

- ✨ **動畫效果**：逼真的抽籤轉盤動畫
- 🎰 **隨機抽取**：從1到22號中隨機抽取一個號碼
- 💨 **加速效果**：動畫會逐漸加速，增加期待感

### 使用方法

1. 確保安裝了Python 3。
2. 運行以下命令：

   ```bash
   python lottery.py
   ```

3. 按下 Enter 開始抽籤
4. 程式將播放動畫並顯示抽到的號碼

### 程式說明

- `animate_lottery(duration=2.5)`：執行動畫抽籤，duration參數可調整動畫時長
- 號碼範圍：1到22
- 動畫時間：預設2.5秒
